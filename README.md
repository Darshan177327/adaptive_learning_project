# Adaptive Learning System

## Description
This system segments learners based on behavioral features using PCA + K-Modes clustering and detects real-time cognitive state to adapt educational content automatically.

## Features
- Behavior segmentation of learners
- PCA + K-Modes clustering
- Real-time cognitive state detection
- Adaptive content recommendation
- Streamlit User Interface

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- KModes
- Streamlit

## How to Run
1. Clone the repository
2. Install required packages:
   pip install -r requirements.txt
3. Run the UI:
   streamlit run app/app.py
